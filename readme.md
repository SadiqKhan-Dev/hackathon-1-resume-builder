# Hackathon-1 Resume Builder

A complete step-by-step Resume Builder project created for Hackathon-1.
This repository takes beginners from a simple static resume to a fully dynamic, editable, and shareable resume application, deployed online using Vercel.

## 📖 Project Overview

This project is divided into milestones.
Each milestone adds new features so beginners can learn gradually:

Static Resume (HTML + CSS)

Deploy on Vercel

Dynamic Resume (JavaScript/TypeScript)

Editable Resume

PDF Download + Link Sharing

Unique Shareable URL for Every User

This milestone-based structure helps new learners build a real project step-by-step.

## 📁 Folder Structure

| Folder                             | Purpose                                                             |
| ---------------------------------- | ------------------------------------------------------------------- |
| **Milestone 1 – Static Resume**    | Basic static resume using HTML & CSS.                               |
| **Milestone 2 – Deploy Vercel**    | Shows deployment steps & files.                                     |
| **Milestone 3 – Dynamic Resume**   | Converts resume into a dynamic version using JavaScript/TypeScript. |
| **Milestone 4 – Editable Resume**  | Adds edit mode where users can update their data instantly.         |
| **Milestone 5 – PDF & Share Link** | Adds PDF download and shareable link feature.                       |
| **Milestone 5 (Advanced)**         | Generates a **unique path (URL)** for each user.                    |

## 🚀 How to Run Each Milestone
🔹 Milestone 1 (Static Resume)

Open the Milestone 1 folder

Double-click index.html

Resume will open in your browser

## 🔹 Milestone 2 (Deploy on Vercel)

1. Install Vercel CLI

        npm i -g vercel

2. Inside the milestone folder run:

        vercel 

3. Your project is now deployed.

🔹 Milestone 3 (Dynamic Resume – TypeScript)

1. Go to the folder

2. Install dependencies

        npm install 

3. Compile TypeScript

        npx tsc

4. Open index.html in browser
